# ISC Sanitation Platform

Flutter application for a unified India Sanitation Coalition platform that supports ISC teams, CSR organizations, and government agencies through a shared block-level sanitation dashboard.

## Modules

- Overview dashboard
- Block-level view
- Implementation partner view
- Funding gap analysis
- Project and activity tracker
- Reports and export summary
- User and role management
- Partner performance dashboard
- Data entry for sanitation verticals

## Run

```bash
flutter pub get
flutter run -d chrome
```

## Deploy

This repo is configured for GitHub Pages deployment through GitHub Actions.

- Repository URL: `https://github.com/jiteshrustagi-ctrl/ISC-Sanitation_Platform`
- Expected live URL: `https://jiteshrustagi-ctrl.github.io/ISC-Sanitation_Platform/`

After pushing to `main`, GitHub Actions will build the Flutter web app and deploy it to Pages.
